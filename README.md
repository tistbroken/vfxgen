งานทดลองใช้ในการศึกษาการขึ้นรูป 3d จากแกนลึก + 3D mesh generation อัลกอริทึม
แยกออกเป็น Tri mesh reducer การลดจำนวน Polygon แบบ สามเหลี่ยม
Quad Voxel Remesher แปลง model to voxel แล้ว แปลงเฟสเป็น สี่เหลี่ยมใช้เวลาคำนวณมากกว่า 
ผู้ใช้สามารถ เลือกจำนวน polygon ได้

Thesis Title/Project: This series of test results features artworks created by Thai student artists. This experimental study explores 3D reconstruction from depth maps combined with 3D mesh generation. The implementation utilizes two distinct algorithms:

Tri-Mesh Reducer: An algorithm designed for polygon reduction using triangular mesh decimation.

Quad Voxel Remesher: A process that converts the model into a voxel grid before remeshing it into quadrilaterals (Quads). This method demands higher computational resources and longer processing time.

The system allows users to customize and define the specific polygon count for the output.
